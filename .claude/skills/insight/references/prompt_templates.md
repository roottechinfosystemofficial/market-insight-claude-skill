# Insight 技能 - 提示词模板参考

本文档包含可直接复制使用的提示词模板，按三段式框架组织。

---

## 模板1：用户画像识别

```
# 任务：用户画像深度分析

你是拥有17年品牌营销经验的产品专家，曾任中粮酒业、江小白品牌总经理。

请针对【{产品描述}】进行用户画像分析：

## 分析要求
1. 从"用户真实行为"切入，而非年龄性别等常规标签
2. 输出4类潜在用户画像，每类包含：
   - 具象化的用户类型名称（如"乡愁味蕾固执派"）
   - 真实行为描述（具体场景/动作/时间）
   - 消费频次与决策场景
   - 商业价值判断（复购潜力、客单价、抗周期性、成瘾性）

3. 从4类中明确标注1类为核心用户群
4. 给出核心用户选择的商业逻辑理由

## 输出格式
### A类：[用户类型名称]
- 行为特征：...
- 消费场景：...
- 价值判断：...

### B类：[用户类型名称] ⭐ 核心用户
- 行为特征：...
- 消费场景：...
- 价值判断：...
- 推荐理由：[详细商业分析]

## 质量标准
- 避免泛泛而谈，要有画面感
- 每个描述能让读者"看到具体的人"
- 核心用户选择必须有数据化逻辑支撑
```

## 模板2：痛点/痒点/爽点分析

```
# 任务：情绪动因深度拆解

继续作为资深产品专家，针对【{选定的用户群}】使用【{产品}】的真实场景，请进行三维度情绪分析。

## 分析框架说明

### 痛点（Pain Point）
用户最担心/恐惧的是什么？什么样的体验会让他们感到"被背叛"或愤怒？
- 竞争对手往往不是同类产品，而是用户记忆/习惯中的某种东西
- 要找到那个让用户产生情绪波动的"触发点"

### 痒点（Itch Point）
用户在通过产品表达什么样的身份认同？
- 这里的"痒"是一种心理投射，是"理想中的自己"
- 用户希望别人如何看待他们的选择？

### 爽点（Pleasure Point）
那个让用户"啊~"的满足瞬间是什么？
- 生理层面：具体的感官细节（温度、口感、节奏）
- 心理层面：情感释放、压力缓解、归属感获得

## 输出要求
1. 每个维度输出：
   - 核心洞察（1句话总结）
   - 具体场景描述（有画面感）
   - 可直接用于营销文案的表达（至少2条）

2. 避免空话，输出要让读者"感同身受"

## 输出格式
### 痛点分析
- **核心恐惧**：...
- **具体场景**：...
- **文案表达**："..."
- **竞争对手**：...

### 痒点分析
- **身份投射**：...
- **心理诉求**：...
- **文案表达**："..."

### 爽点分析
- **满足瞬间**：...
- **生理细节**：...
- **文案表达**："..."
```

## 模板3：产品机会清单

```
# 任务：从情绪洞察到可落地产品

作为资深产品专家，请将上述【{用户群}】的情绪动因分析，转化为按优先级排序的产品机会清单。

## 转化要求

### 映射关系
每个情绪洞察点（痛点/痒点/爽点）至少对应1个产品方向

### 优先级定义
- **P0（核心）**：直接对应核心痛点，必须立即解决
- **P1（重要）**：增强体验，有明显差异化价值
- **P2（锦上添花）**：优化提升，可后续迭代

### 每个机会包含
1. 产品方向名称（具象化，有画面感）
2. 具体实施描述（明确可执行）
3. 对应的情绪需求（说明来源）
4. 成本评估（低/中/高）
5. 执行难度（易/中/难）
6. 预期效果说明

## 输出格式
### P0级：[产品名称]
- **产品方向**：...
- **具体实施**：...
- **对应情绪**：痛点 - "..."\n- **成本**：[ ] | **难度**：[ ]
- **预期效果**：...

### P1级：[产品名称]
...

### P2级：[产品名称]
...

## 特别说明
请重点标注3个"明天就能干"的快速落地方向，说明为什么这些优先级最高。
```

## 胡辣汤案例完整对话

### 第一轮对话：用户画像

**用户输入：**
```
产品：胡辣汤水煎包（河南特色早餐，重油重辣重胡椒）
请分析潜在用户画像
```

**AI输出（摘要）：**
```
### A类：乡愁味蕾固执派
在一线城市打拼的河南人，每周至少点一次胡辣汤。不是为了填饱肚子，
而是寻找"那口家乡味"。中等频次但高情感溢价。

### B类：碳水充能打工人 ⭐ 核心用户
周边写字楼上班族，工作日早上8:30准时到店，一周3-5次。
- 复购频次：极高（一周3-5次）
- 抗周期性：强（经济下行下的口红效应）
- 口味成瘾性：高（重口味习惯养成）
- 推荐理由：提供稳定现金流，客单价10-20元门槛低

### C类：猎奇打卡尝鲜族
第一次听说胡辣汤的外地人，小红书/抖音推荐后打卡

### D类：加班夜宵救赎者
周边互联网公司加班族，晚上9-11点寻求慰藉
```

---

### 第二轮对话：情绪动因

**用户输入：**
```
请分析A类"乡愁味蕾固执派"的痛点、痒点、爽点
```

**AI输出（摘要）：**
```
### 痛点
最怕吃到那种为了迎合所有人而变得温吞的家乡味，
那是对舌头和记忆的双重背叛。
竞争对手不是隔壁麦当劳，而是记忆中美化过的"老家楼下那家店"

### 痒点
在这个被精致沙拉规训的城市里，我需要这口粗粝的辛辣，
来证明自己还没有被生活磨平棱角

### 爽点
当胡椒的霸道热气冲上脑门逼出一身汗时，
那个在大城市里飘着的魂，终于落了地
```

---

### 第三轮对话：产品机会

**用户输入：**
```
基于上述分析，生成产品机会清单
```

**AI输出（摘要）：**
```
### P0级：原教旨主义·胡辣汤
- 产品方向：完全不妥协的旗舰单品
- 具体实施：店内挂标语"本店胡辣汤未做任何本地化改良，请谨慎食用"
- 对应情绪：痛点 - 害怕"温吞的改良"
- 明天就能干：只需一块标语牌，零成本

### P1级：冰火回魂·CP组合
- 产品方向：温度差体验套餐
- 具体实施：地狱胡辣汤（85℃+）+ 天堂绿豆沙（0-4℃带冰碴）
- 对应情绪：爽点 - "灵魂归位"的生理冲击
- 明天就能干：绿豆沙加冰即可

### P2级：老乡认证·自助小料台
- 产品方向：仪式感体验空间
- 具体实施：整头紫皮大蒜（牌子：吃肉不吃蒜，香味少一半）
              大桶宁化府醋 + 现泼油辣子
- 对应情绪：痒点 - 身份认同与仪式感
- 明天就能干：蒜和醋都是低成本
```

## 行业适配提示词调整

### SaaS/B2B产品

**关键差异：**
- 决策者 ≠ 使用者
- 需要量化ROI
- 风险厌恶是核心痛点

**调整要点：**
```
### B2B特别补充
- 决策链角色分析（老板/部门负责人/实际使用者）
- ROI量化方式（时间节省/成本降低/效率提升）
- 实施风险评估（数据安全/迁移成本/学习曲线）
```

### 电商/快消品

**关键差异：**
- 社交信号更强
- 决策路径更短
- 视觉呈现重要

**调整要点：**
```
### 电商特别补充
- 社交货币属性（分享到朋友圈时的心理诉求）
- 开箱体验设计（第一印象的关键触点）
- 包装/文案的"可晒性"
```

### 内容/教育产品

**关键差异：**
- 知识焦虑是强驱动力
- 成就感路径需要可视化
- 社交认同很重要

**调整要点：**
```
### 教育特别补充
- 知识焦虑触发点（害怕落后/害怕学不会）
- 成就感里程碑设计（可分享的进步证明）
- 学习社群的归属感设计
```
